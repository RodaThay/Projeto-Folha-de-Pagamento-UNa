💰 Sistema de Folha de Pagamento em Java

Este projeto é um sistema simples de folha de pagamento desenvolvido em Java, executado no terminal/console.
Ele permite cadastrar funcionários, listar funcionários cadastrados e gerar uma folha de pagamento com base no cargo de cada funcionário.

O objetivo do projeto é praticar conceitos básicos de programação em Java, como:

Estruturas de decisão (if, switch)
Estruturas de repetição (while, for)
Métodos
Uso de ArrayList
Entrada de dados com Scanner
Organização de lógica em funções

📋 Funcionalidades

O sistema possui um menu interativo com as seguintes opções:

1️⃣ Cadastrar Funcionário

Permite cadastrar um funcionário informando:

Nome
Matrícula ou CPF
Cargo

Cargos disponíveis:

Cargo    Descrição    Cálculo do salário
1    Funcionário Padrão    Salário base
2    Funcionário Comissionado    Salário base + 10% sobre vendas
3    Funcionário de Produção    Salário base + R$16 por item produzido
2️⃣ Listar Funcionários

Exibe todos os funcionários cadastrados com:

Nome
Matrícula / CPF

3️⃣ Gerar Folha de Pagamento

Mostra:

Nome do funcionário
Matrícula
Salário base
Adicionais ou bônus
Salário total

Também exibe:

✅ Total geral da folha de pagamento da empresa
